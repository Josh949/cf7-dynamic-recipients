Extends Contact Form 7 with dynamic recipient routing based on dropdown selections.

== Description ==

Dynamic Recipients for Contact Form 7 allows you to create forms where the recipient email address changes based on the user's selection in a dropdown menu.

= Features =
* Add dynamic recipient dropdowns to your Contact Form 7 forms
* Manage recipient email addresses through an easy-to-use admin interface
* Seamless integration with Contact Form 7
* Clean and user-friendly interface

= Usage =
1. Go to Contact Form 7 > Dynamic Recipients to set up your recipient options
2. Add the [dynamic_recipient] shortcode to your Contact Form 7 form
3. Configure your form as usual

= Requirements =
* WordPress 5.8 or higher
* PHP 8.0 or higher
* Contact Form 7 plugin

== Installation ==

1. Upload the plugin files to '/wp-content/plugins/cf7-dynamic-recipients'
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Contact Form 7 > Dynamic Recipients to configure recipient options

== Frequently Asked Questions ==

= How do I add a dynamic recipient dropdown to my form? =

Simply add the [dynamic_recipient] shortcode to your Contact Form 7 form where you want the dropdown to appear.

= Is Contact Form 7 required? =

Yes, this plugin requires Contact Form 7 to be installed and activated.
